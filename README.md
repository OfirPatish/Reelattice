# Reelattice

**Your local Tesla incident library** — import dashcam and Sentry clips, tag and bundle events into Cases, review every camera angle, and export proof. All offline. No account. No cloud uploads.

**Download:** [reelattice.vercel.app](https://reelattice.vercel.app)  
**Platform:** Windows 10/11 (desktop app)

---

## Why Reelattice

Most Tesla tools help you **watch a clip once**. Reelattice helps you **keep and organize footage over time** — search a growing library, archive old events, group related clips into Cases, and export when you need to share or document an incident.

| Good for | Not designed for |
|----------|------------------|
| Regular imports from TeslaCam USB | One-off viewing with telemetry overlays |
| Insurance / dispute dossiers (Cases) | Full video editor with trim timelines |
| Large libraries (search, bulk tag, archive) | macOS / Linux (Windows today) |

---

## Highlights

- **Import & library** — copy clips locally; search, filter, tag, and archive
- **Cases** — bundle related events into incident folders with notes
- **6-camera review** — synced grid playback; export matches what you see
- **Export** — ZIP (all cameras) or single grid MP4; optional 30s segment trim
- **USB prompt (Windows)** — opt-in notification when a TeslaCam drive is plugged in; never auto-imports
- **Private by default** — everything stays on your PC

---

## Quick start

1. Install from the [website](https://reelattice.vercel.app/download).
2. Open **Import** → select your TeslaCam USB folder (or drop clips on Library).
3. Review events in **Library** → tag, note, or add to a **Case**.
4. Export ZIP or grid MP4 from the event header when you need proof.

Full walkthrough, shortcuts, and storage paths: **[User Guide](./docs/USER_GUIDE.md)**
